# User Manual

## Onboarding

1. Copy `.env.example` to `.env`.
2. Add keys you already have.
3. Run install and launch scripts.
4. Open the web app.
5. Use Command Center first.

## Core Pages

- Command: run agents.
- Agents: see available specialized agents.
- Skills: see available skills.
- Connectors: see which integrations are active.
- Data Lab: parse files/logs/data.
- Terminal: run approved backend commands.
- Artifacts: create output files.
- Library: save useful outputs.

## Security

- Never commit `.env`.
- Terminal commands are guarded.
- External connectors stay locked unless configured.
